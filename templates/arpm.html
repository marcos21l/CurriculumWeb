Aprende a Resolver Problemas - Matemáticas

Este proyecto surgió como concesión de la Cátedra Telefónica en "Sistemas avanzados de interacción para educación digital" (enlace a resolución) y consiste en un prototipo para iPad de una app cuya principal funcionalidad es que sus usuarios, principalmente alumnos de los primeros cursos de Primaria, adquieran conocimientos relativos a la resolución de problemas matemáticas mediante el uso de esta app.

Funcionamiento

El funcionamiento de la app es bastante sencillo.

Al inicar la app, se muestra un menú inicial el cuál te da la posibilidad de elegir el curso del que quieres realizar los problemas. Una vez elegido el curso, puedes elegir el tipo de problema y posteriormente elegir el problema deseado de la lista de problemas, eso si, el problema ha de estar desbloqueado. Para desbloquear un problema debemos haber solucionado todos los problemas anteriores consiguiendo de esta forma que el usuario vaya adquiriendo ciertas competencias en cada problema que en problemas posteriores tendrá que poner en práctica.

Una vez elegido el problema, se nos proporcionará un enunciado al cuál deberemos dar solución. Si a priori el usuario conoce la solución bastará con que la introduzca y compruebe la solución para validar si esta es correcta o no, y en caso de serlo se desbloquee el siguiente problema.
Como será normal, si el usuario no conoce la solución la app guiará a este mediante una serie de pistas a la solucción consiguiendo que el usuario aprenda que partes del enunciado son importantes, en que ha de fijarse, etc.
Además, cada problema cuenta con un diccionario de palabras que explica gráfica y textualmente las palabras que pueden suponer un mayor problema para los usuarios.

Datos Técnicos

  • Prototipo para iPad Pro 13".
  • Desarrollada con Swift 2.0
  • Base de datos en servidor remoto para almacenar progresos.
  • Uso de memoria local para usos sin conexión.
  • Accesos a base de datos mediante SQL.
  • Script del servidor escritos en PHP.
